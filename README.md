# Hjernens 5 om dagen

Hjernens 5 om dagen er en enkel, AI-støttet læringsapp for systematisk og varig utvikling av ordforråd.

Appen gir brukeren fem nøye utvalgte ord per dag.
Hvert ord behandles ett og ett, med aktivt valg før forklaring.
Feil ord repeteres over tid, riktige ord fases gradvis ut.

Målet er forståelse, ikke pugging.

## Prinsipper
- Fast daglig belastning
- Én handling per skjerm
- Ingen spillmekanismer
- Progresjon basert på mestring

## Status
Prosjektet er i tidlig utviklingsfase.
Spesifikasjon og struktur er etablert.
